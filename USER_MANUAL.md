## 1. บทนำ

### 1.1 ภาพรวมระบบ

ระบบสร้างแบบฟอร์มรับบัตร e-WorkPermit เป็นระบบสำหรับจัดการการออกใบรับบัตร Work Permit ให้แก่คนต่างด้าว ประกอบด้วยฟังก์ชันหลักดังนี้:

- สร้างและพิมพ์ใบรับบัตร e-WorkPermit (พร้อม Barcode)
- **ระบบหมวดหมู่ A-Z** พร้อมแถบสี — ค้นหาเอกสารจากกองได้ง่าย
- ติดตามสถานะการพิมพ์และการรับบัตร
- **ใช้ Barcode Scanner** ค้นหาเอกสารได้ทันที
- จัดทำรายงานสรุปรายวันและรายเดือน
- จัดการผู้ใช้งานและสิทธิ์การเข้าถึง

### 1.2 ความต้องการของระบบ

| รายการ | ความต้องการ |
|--------|------------|
| Browser | Chrome, Firefox, Safari, Edge (เวอร์ชันล่าสุด) |
| Internet | ต้องมีการเชื่อมต่ออินเทอร์เน็ต |
| หน้าจอ | แนะนำ 1280x720 ขึ้นไป |
| Printer | เครื่องพิมพ์ A4 (สำหรับพิมพ์ใบรับ) |
| Barcode Scanner | (อุปกรณ์เสริม) สำหรับค้นหาเอกสาร |

### 1.3 URL เข้าใช้งาน

```
https://receipt.fts-internal.com
```

## 2. การเข้าสู่ระบบ

### 2.1 วิธี Login

1. เปิด Browser และไปที่ `https://receipt.fts-internal.com`
2. ระบบจะแสดงหน้า Login

   ![หน้า Login](screenshots/login-page.png)

3. กรอกข้อมูล:
   - **อีเมล (Email):** อีเมลที่ลงทะเบียนไว้
   - **รหัสผ่าน (Password):** รหัสผ่านของคุณ

4. คลิกปุ่ม **"เข้าสู่ระบบ"**

5. หากข้อมูลถูกต้อง ระบบจะพาไปหน้า Dashboard หลัก

### 2.2 การสมัครสมาชิกใหม่

> **หมายเหตุ:** ผู้ใช้ใหม่ต้องรอการอนุมัติจาก Admin ก่อนจึงจะใช้งานได้

1. ที่หน้า Login คลิก **"สมัครใช้งาน"**
2. กรอกข้อมูล:
   - **อีเมล:** อีเมลสำหรับเข้าสู่ระบบ
   - **ชื่อ-นามสกุล:** ชื่อที่จะแสดงในระบบ
   - **รหัสผ่าน:** อย่างน้อย 8 ตัวอักษร
   - **ยืนยันรหัสผ่าน:** กรอกรหัสผ่านอีกครั้ง

3. คลิก **"สมัครสมาชิก"**
4. ระบบจะแจ้ง "สมัครสำเร็จ! กรุณารอการอนุมัติจาก Admin"
5. รอ Admin อนุมัติบัญชี จากนั้นจึงเข้าสู่ระบบได้

### 2.3 การออกจากระบบ

1. คลิกปุ่ม **"ออกจากระบบ"** ที่มุมขวาบน (สีแดง)
2. ระบบจะกลับไปหน้า Login

## 4. การจัดการข้อมูลใบรับบัตร

### 4.1 เพิ่มข้อมูลใหม่

#### ขั้นตอนการกรอกฟอร์ม

![ฟอร์มกรอกข้อมูล](screenshots/form-add-new.png)

1. **วันที่รับบัตร (Date):**
   - ระบบจะกรอกวันที่ปัจจุบันให้อัตโนมัติ (แสดง Badge "อัตโนมัติ")
   - สามารถเปลี่ยนได้โดยคลิกที่ช่องวันที่

2. **ชื่อคนต่างด้าว (Name):** *จำเป็น
   - กรอกชื่อภาษาอังกฤษ
   - ตัวอย่าง: `MR. JOHN SMITH`
   - ระบบจะใช้ตัวอักษรแรกของชื่อจริง (ข้าม prefix เช่น Mr./Mrs.) สำหรับ**หมวดหมู่ A-Z**

3. **หมายเลข SN บัตร (SN Number):**
   - กรอกเลข Serial Number ของบัตร
   - ตัวอย่าง: `1234567890`

4. **เลขที่คำขอ (Request No.):** *จำเป็น
   - กรอกเลขที่คำขอจากระบบ
   - ตัวอย่าง: `B69113800005272`

5. **เลขที่ใบนัดหมาย (Appointment No.):** *จำเป็น
   - กรอกเลขนัดหมายจากระบบ
   - ตัวอย่าง: `1-BKK001022600639-SW`

#### การอัพโหลดรูปบัตร

รูปจะถูก**บีบอัดอัตโนมัติ**ก่อนอัพโหลด (ขนาดสูงสุด 1200px) เพื่อประหยัดพื้นที่

มี 3 วิธี:

**วิธีที่ 1: คลิกเลือกไฟล์**
1. คลิกที่กรอบรูป "คลิกเพื่อเลือกรูป"
2. เลือกไฟล์รูปภาพ (JPG, PNG)
3. รูปจะแสดงในช่อง Preview พร้อมแสดงขนาดไฟล์

**วิธีที่ 2: ลากและวาง (Drag & Drop)**
1. ลากไฟล์รูปจาก Folder
2. วางลงในกรอบรูป
3. รูปจะแสดงอัตโนมัติ

**วิธีที่ 3: วางจาก Clipboard (Ctrl+V)**
1. Copy รูปจากที่อื่น (เช่น Screenshot หรือจาก LINE)
2. คลิกที่กรอบรูป
3. กด `Ctrl+V` (Windows) หรือ `Cmd+V` (Mac)
4. รูปจะแสดงอัตโนมัติ

> **หมายเหตุ:** ระบบรับเฉพาะ JPG และ PNG เท่านั้น ไฟล์ SVG และ HTML ถูกบล็อกเพื่อความปลอดภัย

#### การบันทึกข้อมูล

1. ตรวจสอบข้อมูลในช่อง **"ตัวอย่างใบรับ"** ด้านขวา
   - Preview จะแสดง **ตัวอักษรหมวดหมู่** และ **แถบสี** ตามชื่อ
2. คลิกปุ่ม **"บันทึกข้อมูล"**
3. ระบบจะแจ้ง "บันทึกข้อมูลสำเร็จ"
4. ข้อมูลจะแสดงในตารางด้านล่าง
5. เลข Doc No. (เลขรับที่) จะถูกสร้างอัตโนมัติ format: `YYYYMMDD-NNN`

### 4.2 แก้ไขข้อมูล

1. ในตารางข้อมูล คลิกปุ่ม **"แก้ไข"** ที่รายการที่ต้องการ
2. ฟอร์มจะเปลี่ยนเป็น **"โหมด: แก้ไข"** (Badge สีส้ม)
3. แก้ไขข้อมูลตามต้องการ
4. คลิก **"บันทึกข้อมูล"**
5. หากต้องการยกเลิก คลิก **"เพิ่มใหม่"** เพื่อกลับโหมดเพิ่มข้อมูล

### 4.3 ลบข้อมูล

> **สิทธิ์:** เฉพาะ **Admin** เท่านั้นที่สามารถลบข้อมูลได้

1. ในตารางข้อมูล คลิกปุ่ม **"ลบ"** ที่รายการที่ต้องการ
2. ระบบจะถามยืนยัน "ต้องการลบข้อมูลนี้?"
3. คลิก **ตกลง** เพื่อยืนยัน หรือ **ยกเลิก**

> Staff และ Manager จะไม่เห็นปุ่มลบ

### 4.4 ค้นหาและกรองข้อมูล

![ค้นหาและกรองข้อมูล](screenshots/search-filter.png)

#### การค้นหา

1. พิมพ์คำค้นหาในช่อง **"ค้นหา"**
2. สามารถค้นหาด้วย:
   - เลขรับที่ (Receipt No.)
   - เลข SN บัตร
   - ชื่อคนต่างด้าว
3. **ค้นหาข้ามวันที่ได้** — ระบบจะค้นจากข้อมูลทั้งหมด ไม่จำกัดเฉพาะวันที่เลือก

#### การค้นหาด้วย Barcode Scanner

1. คลิกที่ช่องค้นหา (ให้ cursor อยู่ในช่อง)
2. ยิง Barcode Scanner ไปที่ barcode บนใบรับ
3. ระบบจะตรวจจับว่าเป็นการ scan (ไม่ใช่พิมพ์มือ) และค้นหาทันที
4. ข้อมูลจะแสดงในตารางทันที

> **เคล็ดลับ:** Barcode Scanner ทำงานได้โดยไม่ต้องกดปุ่มค้นหา — ระบบจะ detect อัตโนมัติ

#### การกรองตามสถานะ

ใช้ Dropdown **"กรองสถานะ"**:

| ตัวเลือก | แสดงรายการ |
|---------|-----------|
| ทั้งหมด | แสดงทุกรายการ |
| พิมพ์แล้ว | รายการที่พิมพ์ใบรับแล้ว |
| รอพิมพ์ | รายการที่ยังไม่พิมพ์ |
| รับบัตรแล้ว | รายการที่มารับบัตรแล้ว |
| รอรับบัตร | รายการที่ยังไม่มารับ |

## 5. ระบบหมวดหมู่ A-Z และแถบสี

### 5.1 หมวดหมู่ A-Z คืออะไร?

เมื่อพิมพ์ใบรับบัตร ระบบจะแสดง **ตัวอักษรตัวใหญ่** มุมขวาบนของใบรับ เพื่อช่วยค้นหาเอกสารจากกองกระดาษได้ง่าย

- ระบบใช้ **ตัวอักษรแรกของชื่อจริง** (ข้าม prefix Mr./Mrs./Miss/Ms.)
- ตัวอย่าง: `MR. JOHN SMITH` → ตัวอักษร **J**
- ตัวอย่าง: `MISS TANAKA YUKI` → ตัวอักษร **T**

### 5.2 แถบสี 5 กลุ่ม

ใบรับจะมี **แถบสีด้านบน** ตามกลุ่มตัวอักษร เพื่อแยกกลุ่มเอกสารด้วยสายตาได้ง่าย:

| กลุ่ม | สี | ตัวอย่างชื่อ |
|-------|------|------------|
| **A-E** | แดง | ANNA, CHEN, DAVID |
| **F-J** | เขียว | FRANK, HIROSHI, JOHN |
| **K-O** | น้ำเงิน | KIMURA, LEE, OMAR |
| **P-T** | ส้ม | PARK, RAVI, TANAKA |
| **U-Z** | ม่วง | UMAR, WANG, YUKI |

![ตัวอย่างใบรับพร้อมหมวดหมู่ A-Z](screenshots/receipt-category-example.png)

### 5.3 แถบสีในตาราง Registry

ตารางข้อมูลจะแสดง **แถบสีด้านซ้าย** ของแต่ละแถว ตามกลุ่มหมวดหมู่เดียวกัน ช่วยให้เห็นการจัดกลุ่มได้ง่าย

### 5.4 วิธีค้นหาเอกสารจากกองกระดาษ

1. **ดูชื่อลูกค้า** — เช่น "MR. HIROSHI TANAKA"
2. **ดูตัวอักษรแรกของชื่อจริง** — H (ข้าม MR.)
3. **หาจากแถบสีเขียว** (กลุ่ม F-J)
4. **หาตัว H** จากกลุ่มสีเขียว → พบเอกสาร

> **เคล็ดลับ:** Batch print จะ **เรียงลำดับ A-Z อัตโนมัติ** ตามชื่อจริง ทำให้กองเอกสารเรียงตามหมวดหมู่พร้อมใช้งาน

## 6. การพิมพ์ใบรับบัตร

### 6.1 พิมพ์รายการเดียว

**วิธีที่ 1: จากฟอร์ม**
1. กรอกหรือเลือกข้อมูลในฟอร์ม
2. คลิกปุ่ม **"พิมพ์ใบรับ"**
3. หน้าต่าง Print จะเปิดขึ้น
4. เลือกเครื่องพิมพ์และตั้งค่า
5. คลิก Print
6. ระบบจะถามยืนยัน "พิมพ์สำเร็จหรือไม่?" → คลิก OK เพื่ออัพเดตสถานะ

**วิธีที่ 2: จากตาราง**
1. ในตารางข้อมูล คลิกปุ่ม **"พิมพ์"** ที่รายการที่ต้องการ
2. หน้าต่าง Print จะเปิดขึ้น
3. พิมพ์ตามปกติ

### 6.2 พิมพ์หลายรายการ (Batch Print)

![Batch Print](screenshots/batch-print.png)

1. ในตาราง ติ๊กเลือกที่รายการที่ต้องการ
   - หรือคลิก **"เลือกทั้งหมด"** ที่หัวตาราง (เลือกเฉพาะหน้าปัจจุบัน)
2. จะแสดงจำนวนที่เลือก เช่น "เลือก 5 รายการ"
3. คลิกปุ่ม **"พิมพ์ที่เลือก"**
4. ระบบจะ **เรียงลำดับ A-Z อัตโนมัติ** ตามชื่อจริง (ข้าม prefix)
5. สร้างหน้าพิมพ์รวมทุกรายการ (แต่ละใบ 1 หน้า A4)
6. พิมพ์ตามปกติ

> **สำคัญ:** Batch print จะเรียง A-Z โดยอัตโนมัติ เอกสารที่ออกมาจะเรียงตามหมวดหมู่ A-Z พร้อมสำหรับค้นหา

## 7. การติดตามสถานะ

### 7.1 สถานะการพิมพ์

| สถานะ | ความหมาย |
|-------|---------|
| พิมพ์แล้ว | ใบรับถูกพิมพ์แล้ว |
| รอพิมพ์ | ยังไม่ได้พิมพ์ใบรับ |

- สถานะจะอัพเดตอัตโนมัติหลังพิมพ์ (เมื่อกดยืนยัน)

### 7.2 สถานะการรับบัตร

| สถานะ | ความหมาย |
|-------|---------|
| รับแล้ว | คนต่างด้าวมารับบัตรแล้ว |
| รอรับ | ยังไม่มารับบัตร |

### 7.3 การเปลี่ยนสถานะรับบัตร

1. ในตาราง คอลัมน์ "รับบัตร"
2. คลิกที่ Badge สถานะ:
   - คลิก **"รอรับ"** → เปลี่ยนเป็น **"รับแล้ว"**
   - คลิก **"รับแล้ว"** → เปลี่ยนกลับเป็น **"รอรับ"** (ยกเลิก)
3. สถานะจะอัพเดตทันทีบน Cloud

> **หมายเหตุ:** สถานะ "พิมพ์" กับ "รับบัตร" เป็นอิสระจากกัน — สามารถ mark ว่ารับบัตรแล้วโดยไม่ต้องพิมพ์ผ่านระบบก็ได้

## 8. สรุปรายวันและการเลือกวันที่

### 8.1 Date Picker — เลือกวันที่ดูข้อมูล

![Date Picker และสรุปรายวัน](screenshots/date-picker-summary.png)

1. ที่ด้านบนของ Summary Cards มี **Date Picker**
2. Default: วันที่ปัจจุบัน
3. คลิกเลือกวันที่อื่นเพื่อดูข้อมูลย้อนหลัง
4. ระบบจะอัพเดต:
   - สรุปรายวัน (Summary Cards)
   - ตารางข้อมูล (แสดงเฉพาะวันที่เลือก)

> **หมายเหตุ:** การค้นหาจะ**ค้นข้ามวันที่**ได้ — เมื่อพิมพ์คำค้นหา ระบบจะค้นจากข้อมูลทั้งหมดไม่จำกัดเฉพาะวันที่เลือก

### 8.2 ดูสถิติประจำวัน

ส่วนสรุปรายวันแสดง 5 Cards:

| Card | แสดงจำนวน |
|------|----------|
| ผลิตบัตรทั้งหมด | จำนวนรายการทั้งหมดของวัน |
| พิมพ์ใบรับแล้ว | จำนวนที่พิมพ์แล้ว |
| รอพิมพ์ใบรับ | จำนวนที่รอพิมพ์ |
| รับบัตรแล้ว | จำนวนที่มารับแล้ว |
| รอรับบัตร | จำนวนที่รอรับ |

### 8.3 Export ข้อมูลรายวัน

> **สิทธิ์:** เฉพาะ Manager และ Admin

**Export CSV:**
1. คลิกปุ่ม **"Export CSV"**
2. ไฟล์ CSV จะถูกดาวน์โหลด
3. เปิดด้วย Excel หรือ Google Sheets

**Export PDF:**
1. คลิกปุ่ม **"Export PDF"**
2. หน้าต่าง Print จะเปิดขึ้น
3. เลือก "Save as PDF"
4. บันทึกไฟล์

## 9. การแบ่งหน้า (Pagination)

### 9.1 ตาราง Registry

เมื่อมีข้อมูลมากกว่า 50 รายการ ระบบจะ**แบ่งหน้าอัตโนมัติ** (50 รายการ/หน้า)

![Pagination](screenshots/pagination.png)

**การใช้งาน:**
- ด้านล่างตาราง จะแสดง **"แสดง 1-50 จาก 250 รายการ"**
- คลิก **"ถัดไป"** หรือ **"ก่อนหน้า"** เพื่อเปลี่ยนหน้า
- คลิกเลขหน้าโดยตรง (เช่น 1, 2, 3, 4, 5)
- เมื่อค้นหาหรือเปลี่ยน filter → ระบบจะ **reset กลับหน้า 1** อัตโนมัติ

**สิ่งที่ควรทราบ:**
- "เลือกทั้งหมด" จะเลือกเฉพาะ **หน้าปัจจุบัน** (50 รายการ) ไม่ใช่ทั้งหมด
- ถ้ามี ≤ 50 รายการ → ไม่แสดง pagination
- Pagination จะ **ไม่ปรากฏ** ในหน้าพิมพ์

### 9.2 Activity Log

Activity Log ก็มี pagination เช่นกัน (50 รายการ/หน้า) ทำงานเหมือนตาราง Registry

## 12. จัดการผู้ใช้

> **สิทธิ์:** เฉพาะ Admin

### 12.1 เข้าสู่หน้าจัดการผู้ใช้

1. คลิกปุ่ม **"จัดการผู้ใช้"** ที่ Header
2. จะแสดงหน้าต่าง User Management

### 12.2 ดูรายชื่อผู้ใช้

ตารางแสดง:
- ชื่อผู้ใช้
- อีเมล
- บทบาท (Role)
- สถานะการอนุมัติ
- วันที่สร้าง

### 12.3 อนุมัติผู้ใช้ใหม่

1. ผู้ใช้ใหม่จะมีสถานะ **"รออนุมัติ"**
2. คลิก **"อนุมัติ"** เพื่ออนุมัติ
3. หรือคลิก **"ปฏิเสธ"** เพื่อปฏิเสธ

### 12.4 เปลี่ยน Role ผู้ใช้

1. คลิก **"แก้ไข"** ที่ผู้ใช้ที่ต้องการ
2. เลือก Role ใหม่: **Staff**, **Manager**, หรือ **Admin**
3. คลิก **"บันทึก"**

### 12.5 Reset Password

1. คลิก **"Reset Password"** ที่ผู้ใช้ที่ต้องการ
2. ระบบจะส่ง email ลิงก์ตั้งรหัสผ่านใหม่ให้ผู้ใช้
3. ผู้ใช้คลิกลิงก์ → ตั้งรหัสผ่านใหม่

## 13. ระบบ Roles และสิทธิ์

### 13.1 ตารางสิทธิ์

| สิทธิ์ | Staff | Manager | Admin |
|-------|:-----:|:-------:|:-----:|
| ดูข้อมูล | ✅ | ✅ | ✅ |
| เพิ่มข้อมูล | ✅ | ✅ | ✅ |
| แก้ไขข้อมูล | ✅ | ✅ | ✅ |
| ลบข้อมูล | ❌ | ❌ | ✅ |
| พิมพ์ใบรับ | ✅ | ✅ | ✅ |
| Export ข้อมูล | ❌ | ✅ | ✅ |
| รายงานรายเดือน | ❌ | ✅ | ✅ |
| Activity Log | ❌ | ❌ | ✅ |
| จัดการผู้ใช้ | ❌ | ❌ | ✅ |

### 13.2 คำอธิบาย Role

**Staff (พนักงาน)**
- สามารถดู เพิ่ม แก้ไข และพิมพ์ข้อมูลได้
- ไม่สามารถลบข้อมูล, Export, หรือดูรายงานเดือน
- เหมาะสำหรับพนักงานปฏิบัติงานประจำวัน

**Manager (ผู้จัดการ)**
- มีสิทธิ์เหมือน Staff + Export + รายงานเดือน
- **ไม่สามารถลบข้อมูล** จัดการผู้ใช้ หรือดู Activity Log
- เหมาะสำหรับหัวหน้างาน

**Admin (ผู้ดูแลระบบ)**
- มีสิทธิ์ทั้งหมดในระบบ รวมถึงลบข้อมูล
- จัดการผู้ใช้, ดู Activity Log, Reset Password
- เหมาะสำหรับผู้ดูแลระบบ

## 14. คำถามที่พบบ่อย (FAQ)

### Q1: ลืมรหัสผ่าน ทำอย่างไร?
**A:** ติดต่อ Admin เพื่อกด Reset Password ให้ — ระบบจะส่ง email ลิงก์ตั้งรหัสผ่านใหม่

### Q2: สมัครแล้วทำไมเข้าไม่ได้?
**A:** ต้องรอ Admin อนุมัติบัญชีก่อน อาจใช้เวลา 1-2 วันทำการ

### Q3: พิมพ์แล้วแต่สถานะไม่อัพเดต?
**A:** ต้องกด **"ยืนยัน"** หลังพิมพ์เพื่ออัพเดตสถานะ ถ้ากดยกเลิกสถานะจะไม่เปลี่ยน ลอง Refresh หน้าเว็บ (F5) หรือตรวจสอบ Internet

### Q4: รูปภาพไม่แสดง?
**A:** ตรวจสอบว่าไฟล์เป็น JPG หรือ PNG (ระบบไม่รับ SVG) และขนาดไม่เกิน 5MB รูปจะถูกบีบอัดอัตโนมัติ

### Q5: Export แล้วภาษาไทยเป็นภาษาต่างดาว?
**A:** เปิดไฟล์ CSV ด้วย Excel และเลือก Encoding เป็น UTF-8

### Q6: ทำไมไม่เห็นปุ่มบางอัน?
**A:** ปุ่มจะแสดงตาม Role ของคุณ — Staff จะไม่เห็นปุ่มลบ, Export, รายงานเดือน (ดูหัวข้อ 13)

### Q7: ข้อมูลหายไปได้ไหม?
**A:** ข้อมูลถูกเก็บบน Cloud (Supabase) มีความปลอดภัยสูง ไม่หายแม้คอมพิวเตอร์เสีย

### Q8: Barcode Scanner ไม่ทำงาน?
**A:** ตรวจสอบว่า cursor อยู่ในช่องค้นหา แล้วยิง scanner ใหม่ ระบบจะตรวจจับ scanner อัตโนมัติ (ไม่ต้องกดปุ่ม Enter)

### Q9: ใบรับพิมพ์ออกมา 2 หน้า?
**A:** ไปที่ตั้งค่าเครื่องพิมพ์ → ปิด "Headers and footers" แล้วพิมพ์ใหม่

### Q10: "เลือกทั้งหมด" เลือกได้แค่ 50 รายการ?
**A:** ถูกต้อง — "เลือกทั้งหมด" จะเลือกเฉพาะหน้าปัจจุบัน (50 รายการ/หน้า) ถ้าต้องการพิมพ์ทั้งหมด ให้เลือกและพิมพ์ทีละหน้า

## 15. การแก้ไขปัญหาเบื้องต้น

### ปัญหา: หน้าเว็บโหลดช้า

**วิธีแก้:**
1. ตรวจสอบ Internet connection
2. ลอง Refresh หน้าเว็บ (F5 หรือ Ctrl+R)
3. ลองเปิดใน Incognito/Private mode
4. Clear browser cache

### ปัญหา: Login ไม่ได้

**วิธีแก้:**
1. ตรวจสอบ Email และ Password ถูกต้อง
2. ตรวจสอบ Caps Lock
3. หากลืมรหัสผ่าน ติดต่อ Admin
4. หากสมัครใหม่ รอการอนุมัติ

### ปัญหา: บันทึกข้อมูลไม่สำเร็จ

**วิธีแก้:**
1. ตรวจสอบว่ากรอกข้อมูลที่จำเป็นครบ (มี * กำกับ)
2. ตรวจสอบ Internet connection
3. ลอง Refresh และกรอกใหม่
4. ถ้าเห็นข้อความ "ข้อมูลไม่ผ่านการตรวจสอบ" → ตรวจสอบว่าไม่มีอักขระพิเศษ (เช่น <, >, script)

### ปัญหา: พิมพ์ไม่ออก

**วิธีแก้:**
1. ตรวจสอบเครื่องพิมพ์เชื่อมต่ออยู่
2. ตรวจสอบกระดาษและหมึก
3. ลองเปิดหน้าพิมพ์ใหม่
4. ลอง "Save as PDF" แล้วพิมพ์จาก PDF

### ปัญหา: รูปภาพไม่แสดงในใบรับ

**วิธีแก้:**
1. ตรวจสอบว่าอัพโหลดรูปแล้ว
2. รูปต้องเป็น JPG หรือ PNG
3. ขนาดไฟล์ไม่ควรเกิน 5MB
4. ลองอัพโหลดรูปใหม่

### ปัญหา: Barcode ไม่แสดงในใบรับ

**วิธีแก้:**
1. ตรวจสอบ Internet — ระบบต้องโหลด Barcode library จาก Internet
2. ถ้า Barcode ไม่โหลด ระบบจะแสดง "Doc No." เป็นข้อความแทน
3. ลอง Refresh หน้าเว็บแล้วพิมพ์ใหม่
